# 贪吃蛇游戏 - 规格文档

## 1. 项目概述

- **项目名称**: 经典贪吃蛇
- **项目类型**: 2D 浏览器游戏
- **核心功能**: 使用方向键控制蛇的移动，吃食物增长，碰撞检测
- **目标用户**: 休闲游戏玩家

## 2. 视觉规格

### 画布设置
- 游戏区域: 400x400 像素 (移动端自适应缩放)
- 网格大小: 20x20 像素 (20列 x 20行)
- 背景色: #1a1a2e (深蓝色)
- 网格线: #16213e (淡蓝色，1px)

### 色彩方案
- 蛇头: #00ff88 (亮绿色)
- 蛇身: #00cc6a (绿色渐变至尾部 #009950)
- 食物: #ff6b6b (珊瑚红)
- 食物光晕: #ff8787 (淡红色，脉冲动画)
- 分数文字: #ffffff (白色)
- 强调色: #ffd93d (金黄色，用于最高分)

### 界面样式
- 字体: "Press Start 2P" (像素风格) 或系统等宽字体
- 开始/结束界面: 半透明黑色遮罩 + 居中文字
- 按钮: 圆角矩形，悬停时放大 1.05 倍

## 3. 游戏机制

### 蛇的行为
- 初始长度: 3 节
- 初始位置: 画布中央
- 初始移动方向: 向右
- 移动速度: 每 120ms 移动一格 (可暂停)
- 转向: 180度反转禁止

### 食物生成
- 随机位置
- 不可出现在蛇身上
- 吃后随机重生

### 得分系统
- 每吃一个食物: +10 分
- 最高分: 使用 localStorage 持久化存储

### 碰撞检测
- 撞墙: 游戏结束
- 撞自己: 游戏结束

## 4. 交互规格

### 键盘控制
- 方向键 (↑↓←→): 控制蛇移动方向
- 空格键: 暂停/继续
- Enter 键: 在开始/结束界面开始新游戏

### 触摸控制 (移动端)
- 滑动方向检测: 根据滑动方向改变蛇的移动方向
- 轻触屏幕: 暂停/继续

## 5. 游戏状态

### 状态机
1. **START**: 显示开始界面
2. **PLAYING**: 游戏进行中
3. **PAUSED**: 游戏暂停
4. **GAME_OVER**: 显示结束界面

### 状态转换
```
START → (按Enter/点击开始) → PLAYING
PLAYING → (按空格/点击) → PAUSED
PAUSED → (按空格/点击) → PLAYING
PLAYING → (撞墙/撞自己) → GAME_OVER
GAME_OVER → (按Enter/点击重新开始) → START
```

## 6. 音效规格 (Web Audio API)

- 吃食物: 短促上升音调 (100ms)
- 游戏结束: 下降音调 (200ms)
- 背景: 无 (保持安静)

## 7. 动画效果

- 食物: 脉冲缩放动画 (scale 0.8-1.2, 500ms)
- 蛇移动: 平滑过渡
- 游戏结束: 蛇身闪烁 3 次
- 分数增加: 数字弹跳动画

## 8. 响应式设计

- 画布最大宽度: 400px
- 移动端: 画布宽度 = 屏幕宽度 - 40px
- 保持正方形比例
- 控制按钮 (移动端): 屏幕底部 4 个方向按钮

## 9. 代码架构

### 类设计
```
Game - 主游戏类
  ├── Snake - 蛇类
  │   └── Segment - 蛇身段
  ├── Food - 食物类
  ├── InputHandler - 输入处理
  ├── AudioManager - 音效管理
  └── UIManager - 界面管理
```

## 10. 验收标准

1. ✅ 蛇可以通过方向键控制移动
2. ✅ 蛇吃到食物后身体增长
3. ✅ 分数实时显示和更新
4. ✅ 最高分正确保存和显示
5. ✅ 撞墙或撞自己后游戏结束
6. ✅ 开始、游戏中、结束界面正常切换
7. ✅ 暂停/继续功能正常
8. ✅ 触摸设备上可正常游戏
9. ✅ 音效正常播放
10. ✅ 视觉效果流畅
